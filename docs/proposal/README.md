## ⚠️ Before making this repo public

`Technical_Proposal.pdf` includes:
- A **Proposed Team** page with full names and roles
- **Experience / past project** pages referencing client organizations (Indian Army, Indian Navy, DRDO) and company branding (Neemus)

If this repository will ever go public, **remove or redact pages 13–16** of the PDF first (or keep this folder out of the public repo entirely and link to it privately). These are fine for an internal/private repo but shouldn't be pushed to a public GitHub repo without checking with whoever owns that client relationship.
